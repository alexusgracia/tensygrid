MTI Toolbox for MATLAB, Version 2.1

The MTI Toolbox for MATLAB provides a suite of tools for working with multilinear time-invariant (MTI) models. Parameters of MTI models can be represented as tensors, i.e., multidimensional or N-way arrays. Tensors offer a wide range of efficient computation and decomposition methods. This toolbox provides many standard methods for modeling, identification, analysis, and controller design of MTI systems.

Downloading

This toolbox is open source and hosted on mti.systems. See LICENSE.txt for the terms of the license (2-clause BSD).

Be advised that a recent version of MATLAB is required. Once you have downloaded or cloned the MTI Toolbox, use addpath to add the mti_toolbox directory to your path. Type doc mti_toolbox to access this documentation. For more information, see README.md

Functionality

The MTI Toolbox provides the classes and functions for manipulating multilinear time-invariant systems using MATLAB's object-oriented features.

  • Multilinear Time-Invariant State-Space Models - The MTI Toolbox supports the creation of multilinear time-invariant state-space models. The MTI systems can be initialized in matrix or tensor form.
  • Descriptor Multilinear Time-Invariant State-Space Models - The MTI Toolbox supports the creation of descriptor multilinear time-invariant state-space models. The MTI systems can be initialized in matrix or tensor form.
